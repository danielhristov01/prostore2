## Bigger ads attract more attention — but at diminishing returns vs. cost

**Impact: LOW** — Empirical fact: bigger ads attract more eyeballs. The Cashvertising data:

| Ad size | Relative attention |
|---|---|
| Quarter-page (control) | 100% |
| Half-page (2× area) | 187.6% |
| Full-page (4× area) | 352% |

So doubling the ad size *almost* doubles attention. Quadrupling the size *more than triples* attention. Returns are positive but sub-linear with respect to area, and (separately) sub-linear with respect to cost — most outlets charge more than 2× for a half-page vs. a quarter-page.

The practical rule: **size is a real lift, but a paid one.** When deciding whether to upgrade ad size, the question is whether the attention lift exceeds the cost premium. For high-intent campaigns (launches, key promotional windows), it usually does. For evergreen / always-on placements, smaller-and-more-frequent often wins.

### How to use the size effect

- **For launches** — go bigger than usual for the first wave. Attention compounds with novelty.
- **For competitive moments** — out-size competitors during high-stakes windows (holiday season, industry events).
- **For positioning** — a brand consistently running half-page where competitors run quarter-page reads as larger / more established.
- **For testing** — when running a new offer, test in larger size first; you want maximum exposure to the message before scaling.
- **Pair with white-space wrapping** (`layout-white-space.md`) — buy the larger size but use white space to isolate the smaller core ad. Compounding effect.

### Common mistakes

- **Assuming size alone fixes a weak ad** — a bigger weak ad is still a weak ad, just more visible.
- **Filling the larger size with more clutter** — bigger should mean more breathing room, not more crammed content.
- **Not testing size against frequency** — same budget split between bigger-less-often and smaller-more-often often produces clearer data than either extreme alone.
- **Forgetting the cost ratio** — outlets don't price linearly. A "2× the size" upgrade is often "3× the price." Know the ratio before defaulting to bigger.

### When to apply

- Print and display ad placement decisions.
- Launch campaigns.
- Competitive windows where dominance matters.
- When budget is genuinely flexible on size.

### When NOT to apply

- When budget is fixed and frequency would be cut to afford size — frequency usually wins for awareness.
- Digital ads where the platform sets size (paid social, search) — focus on creative quality instead.
- When the ad is genuinely strong at smaller size — don't scale a winner unnecessarily.
